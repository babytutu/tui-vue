# tui-vue [<img src="https://img.shields.io/npm/v/tui-vue.svg">](https://www.npmjs.com/package/tui-vue)

mobile components with vue2
## Install

```js
import Vue from 'vue'
import Tui from 'tui-vue'
import 'tui-vue/dist/style.css'
Vue.use(Tui)
```
